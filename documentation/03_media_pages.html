Intro

This section will describe and show the galleries and portfolio sections. These can be added on individual pages or inside other pages that are not fullscreen and require their own content to house the entire element. For example, if you want to set a gallery inside a blog page, you'll have to include it in a container class or if you wish, a content class to exclude it from everything else. 

 

Media

Gallery Round and Square

<div class="gallery gallery-round">
    <a class="show-gallery" href="images/pictures/0.jpg" alt="img" title="Image Caption">
        <img class="preload-image responsive-image" data-original="images/pictures/0t.jpg" alt="img">
    </a>
</div>

The above code generates a simple, thumbnail gallery that will show thumbnails depending on the screen size. 

You can replace gallery-round with gallery-square to change the format of the image. 

To create more gallery items simply duplicate the anchor on line 2 3 4 and replace the href with your image in the anchor and data-original with your image in the img class. Images are preloaded. 

Gallery Blocks

<div class="gallery-blocks">
    <a class="show-gallery" href="images/pictures/0.jpg" alt="img" title="Image Caption">
        <em class="overlay"></em>
        <strong>Breakfast</strong>
        <img class="preload-image responsive-image" data-original="images/pictures/0t.jpg" alt="img">
    </a>
</div>

Gallery blocks will expand to the full width of the screen and popualte with thumbnails. To create more items simply duplicate the anchor inside. Replace the strong class with your own text. To replace the image you'll have to replace the anchor href and the data-original.

Portfolio One & Two

The one and two column portfolio are basically identical. Their housing class determines if the portfolio shows one or two columns

<div class="portfolio portfolio-NUMBER-item">
    <div class="portfolio-item">
        <img class="preload-image responsive-image" data-original="images/pictures/1w.jpg" alt="img">
        <h1>Portfolio Item 1</h1>
        <p>
            Nam dictum, sapien et auctor bibendum, purus lectus porttitor, 
            at venenatis libero nunc non tortor aenean dignissim id auctor. 
        </p>
        <div class="portfolio-icons">
            <a href="#" class="portfolio-icon scale-hover bg-green-dark"><i class="fa fa-link"></i></a>
            <a href="#" class="portfolio-icon scale-hover bg-red-dark"><i class="fa fa-heart"></i></a>
            <a href="#" class="portfolio-icon scale-hover bg-blue-dark"><i class="fa fa-share-alt"></i></a>
        </div>
        <div class="clear"></div>
    </div>
</div>

Replace the portfolio-NUMBER-item at the top with portfolio-one-item or portfolio-two-item. 

Inside this class there will be classes called portfolio-item. Duplicate these to create more items.

Inside the portfolio item classes, you'll locate an image at the top, a title, text, and the icons that can be replaced with your own. The color classes from the first chapter apply here as well.

Portfolio Fluid

To create a fluid portfolio, meaning images will appear in columns as the screen expands use the classes inside the portfolio one and two explained above and rename the main housing class from portfolio-one-item or portfolio-two-item to portfolio-fluid-item 

Portfolio Cards / Wide

This portfolio can store cards with captions on them and no activation, they can only redirect to a differnt page.

<div class="portfolio-wide">

    <div class="portfolio-wide-item portfolio-wide-card">
        <a href="#" class="portfolio-wide-caption">
            <img class="preload-image responsive-image" data-original="images/pictures/0w.jpg" alt="img">
            <h3>Breakfast</h3>
            <em>Card subtitle / description here</em>
            <div class="overlay"></div>
        </a>
    </div>

</div>

Portfolio wide is the wrapper that will hosue all the portfolio-wide-item portfolio-wide-card classes. Duplicate the portfolio-wide-item portfolio-wide-card to create more items. 

The anchor inside will send you to your designated location

The image inside the anchor is the image of the portfolio, replace the data-original with your own image

The H3 is the title of the caption

The em is the subtitle of the caption.

The overlay is under the caption background

And that's about it! duplicate it as you need it! 

Portfolio Wide / Cards

The portfolio wide is similar to the cards, but has one title over the image and then opens down in a toggle like design to insert more content for each image. 

<div class="portfolio-wide">

    <div class="portfolio-wide-item">
        <div class="portfolio-wide-caption">
            <a href="#">
                <img class="preload-image responsive-image" data-original="images/pictures/0ww.jpg" alt="img">
            </a>
            <h3>Breakfast</h3>
            <div class="overlay"></div>
        </div>

        <div class="portfolio-wide-content">
            <!--Portfolio Content When Tapped-->
        </div>
    </div>  

</div>

The portfolio wide has the same wrapping class as the card. Inside this there is only a class called portfolio-wide-item. Duplicate that to create mor portfolios. 

Inside this there is an anchor, in which there is the image you'll have to replace. 

The H3 is the caption title, when you tap it. 

The portfolio-wide-content houses all the text that will show when you tap the content. 

Adaptive Gallery / Portfolio 

The gallery and portfolios that are adaptive rely on the same classes to function. 

<div class="portfolio-adaptive-controls">
    <h4>Change View</h4>                        
    <a class="portfolio-switch adaptive-three" href="#"><i class="fa fa-th"></i></a>
    <a class="portfolio-switch adaptive-two" href="#"><i class="fa fa-th-large"></i></a>
    <a class="portfolio-switch adaptive-one active-adaptive" href="#"><i class="fa fa-navicon"></i></a>
    <div class="clear"></div>
</div>


<div class="content">
    <div class="portfolio-adaptive portfolio-adaptive-one">

        <div class="portfolio-item">
            <!-- Add only Anchor and Image Below for Gallery Mode -->
            <a class="show-gallery" href="images/pictures/1.jpg" title="Caption here!"><img data-original="images/pictures/1.jpg" alt="img" class="preload-image responsive-image"></a>
            <!-- Add the H4 and Paragraph Below for Portfolio Mode -->
            <h4>Item 1</h4>
            <p>
                Lorem ipsum dolor sit amet, consectetur adipiscing elit. Aliquam semper 
                semper tellus. Ut arcu turpis, dignissim sed tortor sed, posuere.
                <a href="#" class="adaptive-more right-text">Read More <i class="fa fa-long-arrow-right"></i></a>
            </p>
        </div>           
    </div>

</div>

Dupllicate the portfolio item to create more items. Remove the H4 and P to use it gallery mode, or leave them there for portfolio mode. 

The first 3 anchors in the portfolio-adaptive-controls are the ones that will toggle the state. By default in the gallery active-adaptive is set to the adaptive-three On the portfolio is set to adaptive-one. 

You''ll also have to set portfolio-adaptive-three / portfolio-adaptive-two / portfolio-adaptive one to the portfolio-adaptive class that will match the controls.
